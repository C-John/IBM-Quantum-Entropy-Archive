# IBM Quantum Entropy Archive
**A Data Engineering Project for High-Entropy Data Ingestion and Archival**

## Project Overview
This project focuses on building a robust **Java-based Cleaning Engine** to process and archive complex datasets into a structured **Data Lake** using Parquet.

## Technical Stack
* **Language:** Java 25 (Trixie)
* **Build Tool:** Maven
* **Environment:** VS Code Dev Containers
* **Version Control:** Git & GitHub

## Architecture
The system utilizes a multi-stage pipeline:
1. **Ingestion:** Raw data collection.
2. **Cleaning:** Java-driven transformation and validation.
3. **Storage:** Archival into a high-performance Parquet-based Data Lake.
